# Pila Tecnológica: Music Downloader

## Lenguaje de Programación

- Python (versión 3.13+)

## Frameworks y Librerías

- **Click**: Utilizado para construir la interfaz de línea de comandos (CLI).
- **Spotipy**: Librería Python para la API de Spotify.
- **yt-dlp**: Herramienta de línea de comandos para descargar videos y audio de YouTube y otros sitios.

## Dependencias del Sistema

- **ffmpeg**: Herramienta esencial para el procesamiento y conversión de archivos multimedia, requerido para la manipulación de audio.
