# Guía de Producto: Music Downloader

## Concepto Inicial

Este proyecto es una herramienta CLI para descargar música de YouTube y Spotify.

## Usuarios Objetivo

Los usuarios principales de esta aplicación son individuos que desean descargar sus propias listas de reproducción para escucharlas sin conexión.

## Objetivos Clave para Próximas Versiones

Los objetivos a corto plazo incluyen:
- La integración de una Interfaz Gráfica de Usuario (GUI) para facilitar la interacción y mejorar la experiencia del usuario.
- La incorporación de nuevas librerías para simplificar el proceso de descarga de listas de reproducción.

## Soporte para Plataformas Musicales

Aunque se considerará la integración con Apple Music en el futuro, la prioridad actual es consolidar y mejorar el soporte existente para YouTube y Spotify, asegurando una funcionalidad robusta y eficiente en estas plataformas.

## Organización de Archivos

La música descargada se organizará por lista de reproducción.

## Formato y Calidad de Salida

La aplicación está diseñada para ofrecer audio de alta calidad, como FLAC o WAV, satisfaciendo las necesidades de los audiófilos.
